<h4 dir="rtl" style="text-align:right; font-family:'Noto Nastaliq Urdu', 'Jameel Noori Nastaliq', serif;">1. طلبہ کے سیکھنے کے مقاصد </h4>

<p dir="rtl" style="text-align:right; font-family:'Noto Nastaliq Urdu', 'Jameel Noori Nastaliq', serif;">طلباء اس قابل ہو جانیں گے:</p>

<ul style="direction:rtl; font-family:'Noto Nastaliq Urdu', 'Jameel Noori Nastaliq', serif;">
  <li dir="rtl" style="text-align:right; font-family:'Noto Nastaliq Urdu', 'Jameel Noori Nastaliq', serif;">طلبہ خط کو ادب کے ایک اہم حصے کی نگاہ سے دیکھ سکیں گے۔</li>
  <li dir="rtl" style="text-align:right; font-family:'Noto Nastaliq Urdu', 'Jameel Noori Nastaliq', serif;">خطوط نویسی کی تاریخ اور اس کی دوسرے جدید کی شکل میں فرق کر پائیں گے۔</li>
</ul>

<h4 dir="rtl" style="text-align:right; font-family:'Noto Nastaliq Urdu', 'Jameel Noori Nastaliq', serif;">2. آلات </h4>

<ul style="direction:rtl; font-family:'Noto Nastaliq Urdu', 'Jameel Noori Nastaliq', serif;">
  <li dir="rtl" style="text-align:right; font-family:'Noto Nastaliq Urdu', 'Jameel Noori Nastaliq', serif;">سفید تخته، مارکر، جهازن، خط</li>
</ul>

<h4 dir="rtl" style="text-align:right; font-family:'Noto Nastaliq Urdu', 'Jameel Noori Nastaliq', serif;">3. تعارف </h4>

<p dir="rtl" style="text-align:right; font-family:'Noto Nastaliq Urdu', 'Jameel Noori Nastaliq', serif;">استاد/استانی طلبہ سے پوچھے گا/گی کہ کیا کبھی انہوں نے کسی کو خط لکھا ہے اور پھر طلبہ سے ان کے خط لکھنے کے تجربے کے بارے میں پوچھا جائے گا۔ پھر استاد جامعہ میں معافی کے لیے لکھے ہوئے ذاتی خط کے بارے میں بتائے گا جو کہ استاد سکتوب علیہ تک نہ پہنچا سکتا تھا۔</p>

<h4 dir="rtl" style="text-align:right; font-family:'Noto Nastaliq Urdu', 'Jameel Noori Nastaliq', serif;">4. سبق کی کارروائیاں</h4>

<p dir="rtl" style="text-align:right; font-family:'Noto Nastaliq Urdu', 'Jameel Noori Nastaliq', serif;">استاد/استانی مندرجہ ذیل سرگرمیوں کے ذریعے سبق کے مقاصد کو حاصل کرنے کی کوشش کرے گا/گی۔</p>

<p dir="rtl" style="text-align:right; font-family:'Noto Nastaliq Urdu', 'Jameel Noori Nastaliq', serif;"><strong>سرگرمی 1: خطوط نویسی کی تاریخ</strong></p>

<ul style="direction:rtl; font-family:'Noto Nastaliq Urdu', 'Jameel Noori Nastaliq', serif;">
  <li dir="rtl" style="text-align:right; font-family:'Noto Nastaliq Urdu', 'Jameel Noori Nastaliq', serif;">استاد/استانی یہ بتائے کی کوشش کرے گا/گی کہ خط لکھنے کا رواج کیوں شروع ہوا۔</li>
  <li dir="rtl" style="text-align:right; font-family:'Noto Nastaliq Urdu', 'Jameel Noori Nastaliq', serif;">دور قدیم میں خط کو ایک جگہ سے دوسری جگہ تک کس طرح سے پہنچایا جاتا تھا۔</li>
  <li dir="rtl" style="text-align:right; font-family:'Noto Nastaliq Urdu', 'Jameel Noori Nastaliq', serif;">خط بھیجے کے لیے گھڑ سوار اور کبوتر کا استعمال کیونکر اور کیسے کیا جاتا تھا۔</li>
</ul>

<p dir="rtl" style="text-align:right; font-family:'Noto Nastaliq Urdu', 'Jameel Noori Nastaliq', serif;"><strong>سرگرمی 2: خطوط کی حالیہ شکل اور چند مثال</strong></p>

<ul style="direction:rtl; font-family:'Noto Nastaliq Urdu', 'Jameel Noori Nastaliq', serif;">
  <li dir="rtl" style="text-align:right; font-family:'Noto Nastaliq Urdu', 'Jameel Noori Nastaliq', serif;">طلبہ کو واضح کیا جائے گا کہ خطوط وقت کے ساتھ اپنی صورت بدلتے ہیں۔</li>
  <li dir="rtl" style="text-align:right; font-family:'Noto Nastaliq Urdu', 'Jameel Noori Nastaliq', serif;">وضاحت غالب یا ابوالکلام آزاد کا ایک خط سنا کر کی جائے گی۔</li>
</ul>

<h4 dir="rtl" style="text-align:right; font-family:'Noto Nastaliq Urdu', 'Jameel Noori Nastaliq', serif;">5. تشخیص </h4>

<ul style="direction:rtl; font-family:'Noto Nastaliq Urdu', 'Jameel Noori Nastaliq', serif;">
  <li dir="rtl" style="text-align:right; font-family:'Noto Nastaliq Urdu', 'Jameel Noori Nastaliq', serif;">خط بهینجی کی دوسری کون سی صورتین بوتنی به، طلبہ اس پر تبادلہ خیال کریں گے۔</li>
  <li dir="rtl" style="text-align:right; font-family:'Noto Nastaliq Urdu', 'Jameel Noori Nastaliq', serif;">واش اپ، انسٹاگرام یا فیسبوک سے کسی ایک خط کو پڑھ کر سنائیں گے۔</li>
</ul>

<h4 dir="rtl" style="text-align:right; font-family:'Noto Nastaliq Urdu', 'Jameel Noori Nastaliq', serif;">6. حوالہ جات </h4>

<ul style="direction:rtl; font-family:'Noto Nastaliq Urdu', 'Jameel Noori Nastaliq', serif;">
  <li dir="rtl" style="text-align:right; font-family:'Noto Nastaliq Urdu', 'Jameel Noori Nastaliq', serif;">خطوط غالب</li>
  <li dir="rtl" style="text-align:right; font-family:'Noto Nastaliq Urdu', 'Jameel Noori Nastaliq', serif;">کتاب: غبار خاطر، ابوالکلام آزاد</li>
  <li dir="rtl" style="text-align:right; font-family:'Noto Nastaliq Urdu', 'Jameel Noori Nastaliq', serif;">کتاب: صوفی کی دنیا، جوشین گارڈر</li>
</ul>
